Flask Boilerplate
=================

##### Contents
- Flask
- Eric Meyer's Reset CSS v2.0
- jQuery
- Jinja2 templating
- social media / SEO `<meta>` tags

#### Use
1. Edit `/templates/base.html` and fill the numbered sections
2. Create a favicon and replace `/static/img/favicon.ico`

#### Directory Structure
```
├── README.md
├── app.py
├── static
│   ├── css
│   │   ├── app.css
│   │   └── reset.min.css
│   ├── img
│   │   └── favicon.ico
│   └── js
│       └── app.js
└── templates
    ├── app.html
    └── base.html
```
